## 🚀 Cara Penggunaan (Sangat Mudah!)

Ikuti 5 langkah sederhana berikut untuk langsung mulai mencetak kwitansi:

### 1. Jangan "Pisahkan" Filenya!
Pastikan Anda dan rekan sejawat tidak memindahkan file `KwitansiBos.exe` keluar dari foldernya. 
> 💡 **Tips Portable:** Jika ingin memindahkan aplikasi ini ke Flashdisk atau laptop lain, Anda wajib menyalin (copy) **seluruh folder "Aplikasi Jadi" secara utuh**.

### 2. Nyalakan Mesin Aplikasi
Klik ganda (*Double-Click*) pada file **`KwitansiBos.exe`**. Sesaat kemudian, jendela hitam Command Prompt (CMD) akan terbuka. 
> ⚠️ **PENTING:** Jangan menutup jendela hitam (CMD) tersebut selama Anda menggunakan aplikasi, karena jendela itu adalah mesin server utamanya!

### 3. Buka Browser Kesayangan Anda
Silakan buka browser andalan yang biasa Anda gunakan di laptop (seperti Google Chrome, Microsoft Edge, Firefox, atau Opera).

### 4. Kunjungi Alamat Aplikasi
Ketik atau klik alamat tautan berikut pada kolom *address bar* browser Anda:
**[http://localhost:9000](http://localhost:9000)**

### 5. Siap Beraksi! 🎉
Selamat! Aplikasi sudah siap digunakan. Sekarang Anda bisa langsung mengatur KOP Surat sekolah, mengunggah file BKU (format PDF), hingga melakukan *restore* atau impor data cadangan (*backup*) berbasis JSON Anda dengan sekali klik.
